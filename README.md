# goit-algo-hw-05

Завдання 2.

 Програма виконує двійковий пошук у відсортованому масиві дробових чисел.
    
    Args:
        arr (list[float]): Відсортований список чисел.
        target (float): Значення для пошуку.
        
    Returns:
        tuple: (кількість_ітерацій, верхня_межа)
               upper_bound - це найменший елемент >= target.
               Якщо такого елемента немає, повертає None.
